Border with content • مستقیم به خود عنصر اعمال می‌شه (داخل خود باکس عنصر). • برای طراحی ظاهری استفاده می‌شه (مثلاً دور دکمه یا کارت یک خط بذاری)-

برای نشون دادن focus یا انتخاب‌شدن یه عنصره (مخصوصاً برای دسترسی بهتر با کیبورد). • اون هم مستقیم روی عنصر قرار می‌گیره ولی بیرون از border ظاهر می‌شه. • معمولاً نازک‌تره، و فقط یه حالت ساده‌ی خط داره.
• یه ویژگی مخصوص Tailwind هست (در CSS عادی نیست). • شبیه به outline هست ولی با استفاده از box-shadow ساخته می‌شه. • بیرون از عنصر نشون داده می‌شه (و می‌تونی رنگ و فاصله و ضخامتش رو کنترل کنی). • برای هایلایت کردن، تاکید تصویری و بازخورد بصری خیلی استفاده می‌شه.
